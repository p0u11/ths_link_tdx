# THS Link TDX (同花顺与通达信联动工具)

本项目实现同花顺(THS)与通达信(TDX)之间的双向联动，支持双向同步显示股票。

## 功能介绍

1.  **同花顺 -> 通达信 (`ths2tdx.py`)**  
    监控同花顺客户端当前显示的股票，自动同步到通达信显示。适用于习惯在同花顺看盘，在通达信交易或分析的场景。

2.  **通达信 -> 同花顺 (`2ths.py`)**  
    通过命令行工具控制同花顺窗口跳转到指定股票。可配置为通达信的外部挂接工具，实现从通达信点击同步到同花顺。

## 环境要求

*   **操作系统**: Windows
*   **Python**: 3.x
*   **依赖库**:
    *   `pymem`: 用于内存读取
    *   `pywin32`: 用于窗口控制

**安装依赖**:
```bash
pip install pymem pywin32
```

## 使用说明

### 1. 同花顺 -> 通达信 (`ths2tdx.py`)

该脚本原理是通过读取同花顺进程的内存来获取当前显示的股票代码。

1.  **获取基址**  
    参考项目目录下的 `同花顺基址.docx` 文档，使用 Cheat Engine 等工具找到同花顺当前浏览股票代码的内存基址偏移量。

2.  **配置脚本**  
    打开 `ths2tdx.py`，找到 main 函数区域，修改 `base_offset` 变量：
    ```python
    # ths2tdx.py
    process_name = "hexin.exe"  # 同花顺进程名
    base_offset = 0x017944D8    # <--- 修改这里为你找到的基址偏移量
    ```

3.  **运行脚本**  
    由于需要读取内存，**必须以管理员身份运行**命令行窗口：
    ```bash
    python ths2tdx.py
    ```

### 2. 通达信 -> 同花顺 (`2ths.py`)

该脚本通过模拟 Windows 消息控制同花顺跳转。

**命令行用法**:
```bash
python 2ths.py <股票代码>
```

**示例**:
```bash
python 2ths.py 600000      # 跳转到浦发银行
python 2ths.py 000001      # 跳转到平安银行
python 2ths.py SZ000001    # 支持带前缀
```

### 3 双向联动 通达信 <-> 同花顺  link.py  

改造点： 通达信和同花顺都采用读取进程内存方式，程序启动默认双向联动，关联股票支持沪深京  

环境依赖：  
当前python版本是3.10.18  其他版本不清楚，运行 python link.py  缺啥装啥  
内存读取方式与这通达信和同花顺版本强相关，如果失效可参考基址查找方法，或者等待link.py更新   

运行说明：  
	程序启动： python link.py  
	启动默认是双向，启动后可在命令行交互，命令如下  
	status  查看当前双向状态  
	enable tdx  打开 通达信=>同花顺  
	disable tdx 关闭  
	enable ths  打开 同花顺=>通达信  
	disable ths  关闭  


## 注意事项

*   **同花顺基址失效**: 同花顺软件更新后，基址偏移量通常会改变，导致 `ths2tdx.py` 失效。此时需要按照文档重新查找基址并更新脚本。
*   **权限问题**: 涉及内存读取和窗口控制，建议全程使用管理员权限。
*   **兼容性**: 本项目仅在 Windows 平台测试通过。
